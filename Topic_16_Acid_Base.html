Acid–Base

KU 2 · ~2 pp
trigger → answer
★ The stepwise approach — do it every time, in order
pH: acidemia (<7.35) or alkalemia (>7.45)?
Primary process: metabolic (HCO₃ moves with pH) or respiratory (pCO₂ moves opposite pH)?
Compensation appropriate? Metabolic acidosis → Winter's: expected pCO₂ = 1.5×HCO₃ + 8 (±2). Over/under-shoot = a second disorder.
Anion gap = Na − (Cl + HCO₃) (normal ~12). Always compute it, even if HCO₃ is normal.
If high gap → delta-delta (ΔAG / ΔHCO₃): ~1–2 pure HAGMA; <1 added NAGMA; >2 added metabolic alkalosis.
If high gap → osmolar gap for toxic alcohols (measured − calculated osm >10).

High anion-gap metabolic acidosis

GOLDMARK / MUDPILES Glycols (ethylene/propylene), Oxoproline, L-/D-Lactate, Methanol, Aspirin, Renal/uremia, Ketoacidosis
Lactic acidosis sepsis/shock, ischemia, metformin
Ketoacidosis DKA, alcoholic, starvation
Osmolar gap >10 → toxic alcohol
Ethylene glycol → oxalic acid → Ca-oxalate crystals, AKI
Methanol → formic acid → blindness
Treat fomepizole + dialysis

Normal anion-gap (hyperchloremic)

Causes GI bicarbonate loss (diarrhea), renal tubular acidosis, carbonic anhydrase inhibitors, saline infusion
Urine anion gap (UNa+UK−UCl): negative = GI loss (kidney appropriately excretes NH₄⁺); positive = renal (RTA)
Renal tubular acidosis
Type 1 (distal) can't excrete H⁺; hypokalemia, urine pH >5.5, stones
Type 2 (proximal) HCO₃ wasting; hypokalemia; Fanconi
Type 4 hypoaldosteronism → hyperkalemia, ↓NH₄⁺

Metabolic alkalosis

Chloride-responsive (urine Cl <20) vomiting/NG suction (HCl loss), diuretics, contraction → corrects with saline
Chloride-resistant (urine Cl >20) hyperaldosteronism, Cushing, Bartter/Gitelman, severe hypokalemia
Maintained by volume depletion, hypokalemia, hyperaldosteronism

Respiratory disorders

Acidosis (↑pCO₂) hypoventilation — COPD, sedation/opioids, neuromuscular, obesity-hypoventilation
Compensation acute HCO₃ ↑1 per 10 ↑pCO₂; chronic ↑~3.5–4 per 10
Alkalosis (↓pCO₂) hyperventilation — anxiety/pain, hypoxia, PE, sepsis, salicylates, pregnancy, high altitude

Mixed disorders (high-yield)

DKA + vomiting HAGMA + metabolic alkalosis (delta-delta >2)
Salicylate toxicity HAGMA + respiratory alkalosis (the classic mixed picture)
COPD + diuretic/vomiting respiratory acidosis + metabolic alkalosis
Sepsis lactic (HAGMA) + early respiratory alkalosis

Pitfalls

Always check compensation over/under-shoot reveals a second disorder
Compute the gap even if HCO₃ normal can hide a mixed HAGMA + alkalosis
Delta-delta unmasks a concurrent metabolic alkalosis (DKA + vomiting)
Salicylates think mixed acid-base, not a single disorder
Osmolar gap for any unexplained HAGMA → toxic alcohol
